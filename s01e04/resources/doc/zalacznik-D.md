# Załącznik D - Kody błędów terminala

## Błędy identyfikacji (E-001 do E-005)

| Kod | Opis | Działanie operatora |
|---|---|---|
| E-001 | Nieczytelny identyfikator nadawcy | Poprosić o ponowne przyłożenie karty; jeśli problem się powtarza - zgłosić przez terminal |
| E-002 | Wygasły identyfikator nadawcy | Poinformować nadawcę o konieczności odnowienia; odmówić przyjęcia przesyłki |
| E-003 | Nadawca zablokowany przez System | Odmówić przyjęcia przesyłki; nie podawać przyczyny |
| E-004 | Nieistniejący odbiorca | Poinformować nadawcę o błędzie w identyfikatorze odbiorcy |
| E-005 | Odbiorca zablokowany przez System | Odmówić przyjęcia przesyłki; nie podawać przyczyny |

## Awarie sprzętu (E-010 do E-014)

| Kod | Opis | Działanie operatora |
|---|---|---|
| E-010 | Awaria modułu wagowego (MW) | Zgłosić incydent przez terminal; wstrzymać przyjmowanie przesyłek |
| E-011 | Awaria modułu wymiarowego (MWy) | Zgłosić incydent przez terminal; wstrzymać przyjmowanie przesyłek |
| E-012 | Awaria modułu rentgenowskiego (MR) | Zgłosić incydent przez terminal; wstrzymać przyjmowanie przesyłek; ewakuować strefę AK |
| E-013 | Awaria modułu spektrometrycznego (MS) | Zgłosić incydent przez terminal; wstrzymać przyjmowanie przesyłek |
| E-014 | Awaria modułu plombującego (MP) | Zgłosić incydent przez terminal; wstrzymać przyjmowanie przesyłek |

## Problemy systemowe

| Kod | Opis | Działanie operatora |
|---|---|---|
| E-020 | Brak łączności z Systemem | Przejść w tryb oczekiwania; odmawiać przyjmowania przesyłek do czasu przywrócenia łączności |
| E-021 | Niestabilna łączność z Systemem | Kontynuować pracę ze zwiększoną ostrożnością; dokumentować wszystkie czynności |
| E-030 | Brak dostępnych wagonów | Poinformować nadawców o opóźnieniu; przyjmować zgłoszenia z szacowanym terminem |
| E-040 | Zablokowana trasa docelowa | Nie przyjmować przesyłek na daną trasę; poinformować nadawców |
| E-099 | Błąd krytyczny | Natychmiast powiadomić Dyspozytora Lokalnego; wstrzymać wszelką działalność stacji |
